⚙️ Configura Supabase

Para activar el panel, necesitas crear un proyecto gratuito en Supabase y completar estos pasos:

1
Crea tu proyecto
Ve a supabase.com → New Project. Guarda tu URL y Anon Key.
2
Crea las tablas en SQL Editor
Copia y ejecuta este SQL en tu proyecto:
-- Tabla de productos CREATE TABLE products ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, name text NOT NULL, brand text NOT NULL, category text NOT NULL, price numeric NOT NULL, stock integer DEFAULT 0, description text, image_url text, is_new boolean DEFAULT false, is_active boolean DEFAULT true, created_at timestamptz DEFAULT now() ); -- Tabla de pedidos CREATE TABLE orders ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, items jsonb NOT NULL, total numeric NOT NULL, status text DEFAULT 'pending', customer_phone text, notes text, created_at timestamptz DEFAULT now() ); -- Storage bucket para fotos INSERT INTO storage.buckets (id, name, public) VALUES ('product-images', 'product-images', true); -- Política para subir imágenes (autenticados) CREATE POLICY "Auth upload" ON storage.objects FOR INSERT TO authenticated WITH CHECK (bucket_id = 'product-images'); CREATE POLICY "Public read" ON storage.objects FOR SELECT TO public USING (bucket_id = 'product-images'); -- Row Level Security ALTER TABLE products ENABLE ROW LEVEL SECURITY; ALTER TABLE orders ENABLE ROW LEVEL SECURITY; CREATE POLICY "Public read products" ON products FOR SELECT TO public USING (is_active = true); CREATE POLICY "Auth manage products" ON products FOR ALL TO authenticated USING (true); CREATE POLICY "Auth manage orders" ON orders FOR ALL TO authenticated USING (true);
3
Edita el archivo admin.html
Reemplaza en el <script> al fondo:
SUPABASE_URL y SUPABASE_KEY con tus credenciales.
4
Crea tu usuario admin
En Supabase → Authentication → Users → Invite User. Usa ese email y contraseña para ingresar.
